# HangmanJSTL
 路Play the game and store your results in a session.      路Generate random words from a XML file      路See your previous score      路I designed the website with CSS3

# TO RUN THIS PROJECT:
  -->Install Tomcat <br>
  -->Add jstl jar to Tomcat lib directory <br>
  -->Copy this file to Tomcat webapps directory <br>
  -->Turn on Tomcat <br>
  -->Access to localhost:yourport/HangmanJSTL/  (for example: http://localhost:8082/HangmanJSTL/) <br>
  -->Enjoy the web app! <br>
