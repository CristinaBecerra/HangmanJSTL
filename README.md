# HangmanJSTL
 路Play the game and store your results in a session.   <br>
 路Generate random words from a XML file  <br>
 路See your previous score   <br>
 路Choose new word when you win (from the ones you haven't play already) <br>

# TO RUN THIS PROJECT:
  -->Install Tomcat <br>
  -->Add jstl jar to Tomcat lib directory <br>
  -->Copy this file to Tomcat webapps directory <br>
  -->Turn on Tomcat <br>
  -->Access to localhost:yourport/HangmanJSTL/  (for example: http://localhost:8082/HangmanJSTL/) <br>
  -->Enjoy the web app! <br>
